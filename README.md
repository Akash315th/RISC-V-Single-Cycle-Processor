# RISC-V Single Cycle Processor (Verilog)

## 🔹 Overview

This project implements a 32-bit Single Cycle RISC-V Processor using Verilog HDL. Each instruction is executed in a single clock cycle.

## 🔹 Features

* 32-bit RISC-V Architecture
* Single cycle execution
* ALU operations (ADD, SUB, AND, OR, SLT)
* Instruction Fetch, Decode, Execute
* Register File and Control Unit implementation

## 🔹 Modules

* Program Counter (PC)
* Instruction Memory
* Register File
* ALU
* Control Unit
* Data Memory
* Multiplexers

## 🔹 Simulation

The design is verified using a testbench and waveform analysis.

## 🔹 Tools Used

* Verilog HDL
* Icarus Verilog
* GTKWave
